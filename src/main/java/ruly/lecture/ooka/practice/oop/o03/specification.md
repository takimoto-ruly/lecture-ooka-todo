### 【概要】
複数の異なるキャラクターを1つのリスト（または配列）にまとめ、多態性を用いて一括で「攻撃」の命令を出すプログラムを構築します。個別のクラスを意識せずに、親クラスの型としてまとめて扱う便利さを体感します。

### 【仕様】
1. **親クラス（一般的なキャラクターを表すクラス）**
   - **メソッド**: 文字列「攻撃した！」を戻り値（`String`）として返す。
2. **子クラスA（戦士を表すクラス）**
   - 親クラスを継承し、攻撃メソッドをオーバーライドする。
   - **戻り値**: 文字列「剣で斬る！」を返す。
3. **子クラスB（魔法使いを表すクラス）**
   - 親クラスを継承し、攻撃メソッドをオーバーライドする。
   - **戻り値**: 文字列「魔法を放つ！」を返す。
