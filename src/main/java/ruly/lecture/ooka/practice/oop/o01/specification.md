### 【概要】
共通の属性（データ）と基本的な振る舞いを持つ「動物」を親クラスとし、それを引き継いだ特定の動物（犬）を子クラスとして定義してください。親クラスの資産を子クラスが再利用する「継承」の基本構造を学びます。

### 【仕様】
1. **親クラス**
   - **フィールド（カプセル化すること）**: 動物の「名前」（文字列型）
   - **コンストラクタ**: インスタンス化の際に「名前」を引数で受け取り、フィールドを初期化する。
   - **メソッド1（getter）**: フィールドの「名前」を外部から取得する。
   - **メソッド2**: 実行すると、「[名前]は眠っています」というメッセージを標準出力（`System.out.println`）に表示する。
2. **子クラス**
   - 親クラスを継承して作成する。
   - **固有のメソッド**: 実行すると、「ワンワン！」というメッセージを標準出力に表示する。
