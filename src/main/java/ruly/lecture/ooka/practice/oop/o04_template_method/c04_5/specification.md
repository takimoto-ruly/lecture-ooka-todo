## レポート出力（実戦的な文字列連結）

### 【概要】
これまでの知識の総まとめです。標準出力（画面表示）ではなく、メソッドの戻り値として「ヘッダー ➔ 本文 ➔ フッター」を連結した1つのレポート（文字列）を完成させて返すプログラムを作成します。

### 【仕様】
1. **親クラス（抽象クラス）**
   - **レポート生成メソッド**: 戻り値 `String`。変更禁止（`final`）とする。
   - 処理内容：`【ヘッダー（抽象）】 + 本文（共通） + 【フッター（抽象）】` を改行（`\n`）で連結して返す。
   - 本文（共通）：どのレポートも共通で「--- 2026年度 中間報告 ---」という文字列にする。
2. **子クラスA（社内用レポート）**
   - ヘッダーに「【社内親睦会】」、フッターに「以上（回覧してください）」を返すように実装。
3. **子クラスB（提出用レポート）**
   - ヘッダーに「【株式会社〇〇 御中】」、フッターに「ご査収のほどよろしくお願い申し上げます。」を返すように実装。
