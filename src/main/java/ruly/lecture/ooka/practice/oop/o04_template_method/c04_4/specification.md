## ゲームのターン処理（引数の活用）

### 【概要】
RPGなどの対戦ゲームにおける「1ターンの流れ」を共通化します。味方と敵のステータス（攻撃力や防御力）をメソッドの「引数」として子クラスに渡す設計を学びます。

### 【仕様】
1. **親クラス（抽象クラス）**
   - **ターン実行メソッド**: 引数に「攻撃力(int)」と「敵の防御力(int)」を受け取る。内部で**【ダメージ計算メソッド（抽象）】**にこの2つの数値を渡し、返ってきたダメージを使って「敵に [ダメージ] の被害を与えた！」と出力する。
2. **子クラスA（戦士のターン）**
   - ダメージ計算を上書き：「攻撃力 - 防御力」の結果をそのままダメージとして返す。
3. **子クラスB（魔法使いのターン）**
   - ダメージ計算を上書き：魔法は防御力を無視するため、「攻撃力」の数値をそのままダメージとして返す。
