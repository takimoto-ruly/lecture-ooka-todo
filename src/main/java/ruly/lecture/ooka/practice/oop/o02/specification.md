### 【概要】
「描画する」という共通の命令（メソッド）に対して、図形の種類（円、四角）ごとに異なる結果を返す仕組みを構築します。同じ名前のメソッドでも、インスタンスの実体によって振る舞いが変わる多態性の基礎を学びます。

### 【仕様】
1. **親クラス（一般的な図形を表すクラス）**
   - **メソッド**: 文字列「図形を描画します」を戻り値（`String`）として返す。
2. **子クラスA（円を表すクラス）**
   - 親クラスを継承し、描画メソッドを上書き（オーバーライド）する。
   - **戻り値**: 文字列「円を描画します」を返す。
3. **子クラスB（四角を表すクラス）**
   - 親クラスを継承し、描画メソッドを上書き（オーバーライド）する。
   - **戻り値**: 文字列「四角を描画します」を返す。
