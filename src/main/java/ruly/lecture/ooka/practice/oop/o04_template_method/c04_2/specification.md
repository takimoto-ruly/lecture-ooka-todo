## 料理レシピ（値の返却と処理の具体化）

### 【概要】
料理を作る手順「材料を切る ➔ 炒める ➔ 味付けをして完成」を共通化します。今回は、最後の「味付けメッセージ」を子クラスから文字列（`String`）として親クラスに返却する形に挑戦します。

### 【仕様】
1. **親クラス（抽象クラス）**
   - **調理メソッド**: 「材料を切りました。」「炒めました。」を順に標準出力へ表示し、最後に**【味付けメソッド（抽象）】から受け取った文字列**を使って「[味付け]をして完成です！」と出力する。
   - **味付けメソッド（抽象）**: 戻り値は `String`。
2. **子クラスA（カレー職人）**
   - 味付けメソッドを上書きし、「カレールーを投入」という文字列を返す。
3. **子クラスB（チャーハン職人）**
   - 味付けメソッドを上書きし、「塩コショウを少々」という文字列を返す。
