## 挨拶ロボット（スケルトンあり・超基本）

### 【概要】
ロボットに挨拶をさせます。「電源ON → 各国語の挨拶 → 電源OFF」という一連の流れ（テンプレート）を親クラスで定義し、挨拶の部分だけを子クラスで自由に切り替えられるようにします。

### 【仕様】
1. **親クラス（抽象クラス）**
   - **全体の流れを制御するメソッド**: 「電源ON」を出力 ➔ **【個別挨拶（抽象メソッド）】を呼び出す** ➔ 「電源OFF」を出力、という順番で実行する。
   - **個別挨拶メソッド（抽象メソッド）**: 戻り値・引数なし。処理は子クラスに強制する。
2. **子クラスA（日本語ロボット）**
   - 個別挨拶メソッドを上書きし、「こんにちは！」と出力する。
3. **子クラスB（英語ロボット）**
   - 個別挨拶メソッドを上書きし、「Hello!」と出力する。
