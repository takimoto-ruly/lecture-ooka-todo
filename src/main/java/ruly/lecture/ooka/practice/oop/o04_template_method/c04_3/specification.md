## ハンバーガーショップ（フックメソッドの導入）

### 【概要】
ハンバーガーの調理手順「下のバンズ ➔ 具材 ➔ ソース ➔ 上のバンズ」を共通化します。さらに、「トッピング（チーズなど）を追加するかどうか」を子クラスがYes/Noで判断できる仕組み（フックメソッド）を取り入れます。

### 【仕様】
1. **親クラス（抽象クラス）**
   - **調理メソッド**: 下のバンズ、具材（抽象）、ソース（抽象）に関するメッセージを順に出力。その後、**【トッピングが必要か？】というメソッド（フック）が `true` を返した場合のみ**「チーズを追加します」と出力し、最後に上のバンズのメッセージを出力する。
   - **トッピングが必要か？（フックメソッド）**: デフォルトでは `false` を返すように親クラスで実装しておく。
2. **子クラスA（普通のハンバーガー）**
   - 具材とソースのメッセージを返すように実装。トッピングは不要なのでフックメソッドは触らない。
3. **子クラスB（チーズバーガー）**
   - トッピングが必要か？のメソッドを上書きし、常に `true` を返すようにする。
